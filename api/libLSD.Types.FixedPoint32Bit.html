Struct FixedPoint32Bit

A 32-bit fixed point number as used in PSX hardware.

In the form:

    MSB ------------------------------- LSB
    SIII IIII IIII IIII IIII MMMM MMMM MMMM

Where S is sign bit, I are integral bits, and M are mantissa bits.
